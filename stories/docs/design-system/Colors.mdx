## Accessibility

All color combinations meet WCAG 2.1 Level AA contrast requirements for normal text (4.5:1) and large text (3:1). The `-on` suffix indicates the appropriate text color for a given background.

**Example:**
- `bg-primary` with `text-primary-on` ensures proper contrast
- `bg-error` with `text-error-on` ensures readable error messages

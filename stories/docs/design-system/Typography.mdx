# Typography

Typography in BCT UI uses Lato as the primary sans-serif font and Georgia for serif. Text sizes range from 10px to 36px, with font weights from 100 to 900. Responsive typography classes ensure readability across devices.

## Best Practices

1. **Use semantic classes**: Prefer `.heading1`, `.body`, etc. over manual size/weight combinations
2. **Maintain hierarchy**: Use heading levels in order (h1 ‚Üí h2 ‚Üí h3)
3. **Limit font weights**: Stick to normal (400), medium (500), and bold (700) for most use cases
4. **Consider line length**: Keep body text between 50-75 characters per line for readability
5. **Test responsiveness**: Verify typography scales appropriately across all screen sizes

## Accessibility

- Minimum font size of 14px (text-sm) for body text
- Sufficient contrast ratios (4.5:1 for normal text, 3:1 for large text)
- Clear visual hierarchy through size and weight
- Responsive scaling ensures readability on all devices

## Key Features

- **Consistency**: Mandatory design tokens ensure identical appearance across applications
- **Developer Experience**: Easy component addition with `npx bct add <component>`
- **Framework Support**: Works with Vite + React Router and Next.js + App Router
- **Type Safety**: Full TypeScript support with proper prop interfaces
- **Local Ownership**: Teams customize components while maintaining design system alignment
- **Accessibility**: Built on Base UI primitives for robust accessibility

## Quick Start

### 1. Install CLI

```bash
npm install -g @bctechnology/ui
```

### 2. Initialize Project

```bash
cd your-project
npx bct init
```

### 3. Add Components

```bash
npx bct add button
npx bct add input
```

### 4. Develop

```bash
npm run dev
```
